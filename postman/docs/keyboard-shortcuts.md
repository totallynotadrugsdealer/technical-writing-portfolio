# Purpose

Postman provides keyboard shortcuts that help you navigate the application more efficiently and perform common actions more quickly.

This guide lists the shortcuts most useful for beginners. Some shortcuts differ slightly between Windows/Linux and macOS.

---

# General Shortcuts

| Action | Windows / Linux | macOS |
|--------|-----------------|-------|
| Open Search | **Ctrl + K** | **⌘ + K** |
| Open Quick Switch | **Ctrl + Shift + K** | **⌘ + Shift + K** |
| Open Settings | **Ctrl + ,** | **⌘ + ,** |
| Open Keyboard Shortcuts | **Ctrl + Shift + ?** | **⌘ + Shift + ?** |

---

# Request Shortcuts

| Action | Windows / Linux | macOS |
|--------|-----------------|-------|

| Save Request | **Ctrl + S** | **⌘ + S** |

| Close Current Tab | **Ctrl + W** | **⌘ + W** |

# Tips

- You can view the complete list of shortcuts at any time by opening the **Keyboard Shortcuts** window from within Postman.
- Most common shortcuts follow the same conventions used by other desktop applications.
- Learning just a few shortcuts—such as **Send Request**, **Save Request**, and **Search**—can significantly speed up your workflow.

---

# Frequently Used Shortcuts

If you're new to Postman, these are the shortcuts you'll probably use most often:

| Task | Shortcut |
|------|----------|
| Send a request | **Ctrl / ⌘ + Enter** |
| Save a request | **Ctrl / ⌘ + S** |
| Search | **Ctrl / ⌘ + K** |
| Find in response | **Ctrl / ⌘ + F** |
| New request | **Ctrl + Alt + N** (Windows/Linux) / **⌘ + Option + N** (macOS) |

---

# Summary

Keyboard shortcuts allow you to work more efficiently in Postman by reducing the need to navigate menus.

As you become more familiar with Postman, using shortcuts for common tasks such as creating requests, sending requests, searching, and saving can significantly improve your productivity.

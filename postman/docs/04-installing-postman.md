# Purpose

This guide explains how to download, install, and launch the Postman desktop application. After completing this guide, you will have Postman installed and ready to begin the initial setup process.

---

# Prerequisites

Before you begin, ensure that you have:

- A stable internet connection.
- Access to an email account.
- A supported operating system (macOS, Windows, or Linux).

---

# Download the Postman application

1. Open your preferred web browser.

2. Navigate to:

   ```
   https://www.postman.com
   ```

3. On the Postman homepage, select **Product** from the navigation bar.

   ![Postman homepage](../images/postman-homepage.png)

4. From the drop-down menu, select **Download Postman**.

   ![Download Postman menu option](../images/download-postman-menu-option.png)

5. The **Download Postman** page opens.

6. Select the download option that matches your operating system.

   Examples include:

   - Apple Silicon Chip
   - Mac Intel Chip
   - Windows (x64)
   - Windows (ARM64)
   - Linux

   ![Download Postman page](../images/download-postman-landing-page.png)

---

# Request a download link

1. Select the appropriate download button for your operating system.

   For example, on an Apple Silicon Mac, select **Apple Silicon Chip**.

2. A dialog box appears asking for your email address.

3. Enter an email address that you can access on the device where you plan to install Postman.

4. Select **Submit**.

   ![Email address dialog](../images/download-email.png)

> **Note**
>
> Postman sends a download link to your email address instead of immediately downloading the application.

---

# Retrieve the download link

After submitting your email address, the download page displays a confirmation message indicating that a download link has been sent.

The message is similar to:

> **Check your inbox**
>
> We sent a download link to your email address. The link typically arrives within one minute.

1. Open your email inbox.

2. Locate the email with the subject:

   ```
   Your Postman download link
   ```

3. Verify that the sender is:

   ```
   noreply@getpostman.com
   ```

   ![Download email](../images/email-download-zip.png)

---

# Download the application

1. Open the download email.

2. Select **Download Postman for macOS** (or the equivalent button for your operating system).

The Postman application begins downloading automatically.

> **Note**
>
> If you selected the wrong operating system earlier, the email also provides download links for other supported platforms, including:
>
> - Mac (Intel Chip)
> - Windows (x64)
> - Windows (ARM64)
> - Linux (ARM64)
> - Linux (x64)
> - Postman Web

---

# Open the downloaded application

When the download has finished, open the downloaded Postman application.

On macOS, a security prompt appears asking whether you want to open an application downloaded using Safari.

Select **Open** to continue.

![Security confirmation dialog](../images/postman-install-dialogue.png)

---

# Move Postman to the Applications folder

After opening Postman for the first time, another dialog recommends moving the application to the **Applications** folder.

The dialog explains that moving the application ensures that future updates are installed correctly.

Select **Move to Applications Folder**.

![Move to Applications Folder dialog](../images/move-app.png)

> **Recommendation**
>
> Although you can select **Do Not Move**, moving Postman to the **Applications** folder is recommended because it allows the application to manage future updates correctly.

---

# Launch Postman

After Postman starts, the welcome screen is displayed.

From this screen, you can:

- Create a free account.
- Log in with an existing account.
- Continue with Google.
- Continue with GitHub.
- Use Single Sign-On (SSO).
- Continue without an account.

![Postman welcome screen](../images/postman-app-homepage.png)

If you are new to Postman, creating a free account allows you to synchronize your work across devices and collaborate with other users.

If you prefer to explore the application before creating an account, select **Continue without an account**.

---

# Verification

Verify that Postman has been installed successfully.

You should be able to:

- Launch the Postman application.
- View the Postman welcome screen.
- Access the available sign-in options.

If the welcome screen is displayed, the installation was successful.

---

# Summary

You have downloaded, installed, and launched the Postman desktop application.

You are now ready to create or sign in to a Postman account and begin using the application.
